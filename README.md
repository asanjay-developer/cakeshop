# 🎂 CakeShop Website

A modern and responsive **Cake Shop Website** built using HTML, CSS, and JavaScript.
This project showcases a clean UI design for a bakery/cake business with smooth layout and attractive visuals.

---

## 🚀 Live Demo

🔗 https://asanjay-developer.github.io/cakeshop/

---

## 📌 Project Overview

The CakeShop website is designed to present delicious cakes in an elegant and user-friendly way.
It includes sections like home, products, about, and contact to give users a complete browsing experience.

---

## ✨ Features

* 🎨 Clean and modern UI design
* 📱 Fully responsive (mobile-friendly)
* 🍰 Product showcase section
* 📖 About section for business details
* 📞 Contact section
* ⚡ Smooth scrolling and simple interactions

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript** – Interactivity
